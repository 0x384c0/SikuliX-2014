View the corresponding JavaDocs of selected classes (relevant for API usage)
The only file you need to download is:
sikulixsetup-${sikuli.usedversion} (store and run in an empty folder)

The following jars would be downloaded automatically while running setup.
The downloadpages are the OSSRH Snapshot repository:
https://oss.sonatype.org/content/groups/public/com/sikulix/
and the MavenCentral repository:
http://repo1.maven.org/maven2/
and for the tessdata stuff:
http://tesseract-ocr.googlecode.com/files/

So you only would need to manually download any of those, if you have problems with setup's inline download.
If you download manually by clicking on the below links, the jars have to be moved either
   -- to a folder Downloads in the folder containing sikulixsetup-X.Y.Z.jar or    -- to the same folder as sikulixsetup-X.Y.Z.jar
Setup will find the local files and ask you wether to use them instead of downloading fresh versions.

BE AWARE: These files are not useable for any other purpose than being processed by setup
... and will be moved during setup into the Sikulix application folder down in your home folder (see docs).
In any case, when rerunning a setup, already downloaded files will always be detected and you are asked wether to reuse.

downloads for setup option 1 (IDE)

sikulixsetupIDE-1.1.0-YYYYMMDD.SSSSSS-NN-forsetup.jar to get the SikuliX IDE
sikulixsetupAPI-1.1.0-YYYYMMDD.SSSSSS-NN-forsetup.jar (API - SikuliX core features)

  ... to edit/run scripts with the IDE you must have at least one of the following (suboptions 1.1 and 1.2):
   Jython 1.1 for scripting with Python/Jython (is Jython 2.7 and needs Java 7+)
  for use with Java 6 download Jython 2.5

   JRuby 1.2 for scripting with Ruby/JRuby (is JRuby 1.7.18)

download for setup option 2 (API)

sikulixsetupAPI-1.1.0-YYYYMMDD.SSSSSS-NN-forsetup.jar For the use of SikuliX with Java and Java aware languages/frameworks

download in any case

you need the libs package for your system
- for Windows: sikulixlibswin
- for Mac: sikulixlibsmac
- for Linux: sikulixlibslux

download for setup option 3

you need the tessdata base english package
- tessdata-eng: tesseract-ocr-3.02.eng.tar.gz
  (leave it as is (....tar.gz) - do not uncompress/unpack)
